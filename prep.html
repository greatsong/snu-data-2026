강의 전에 계정 3종 만들기

아래 3개만 미리 만들어 오면 첫날부터 바로 만들기에 들어갈 수 있어요. 전부 무료이고, 설치할 프로그램은 없습니다. 브라우저(크롬 권장)만 있으면 됩니다.

✅ 오늘 만들 계정 3개
  • ① GitHub — 내가 만든 것을 세상에 올려두는 ‘창고 + 집’
  • ② Streamlit Community Cloud — 코드를 진짜 ‘웹앱’으로 켜 주는 곳 (GitHub로 로그인)
  • ③ AI 서비스 — 코드를 대신 짜 줄 ‘요리사’ (Claude · ChatGPT · Gemini 중 하나)
⚠️ 딱 하나만 부탁드려요

가입할 때 쓴 이메일과 비밀번호를 꼭 메모해 두세요. 특히 ②는 반드시 ①과 같은 GitHub 계정으로 로그인합니다. 세 계정의 이메일을 같게 맞추면 훨씬 편해요.

① GitHub 계정 만들기

GitHub(깃허브)는 내가 만든 파일을 보관하고, 그대로 인터넷 사이트로 띄워 주는 곳이에요.

  1. github.com 접속
    주소창에 github.com 을 치고 들어가, 오른쪽 위 Sign up(가입)을 누릅니다.
  2. 이메일 · 비밀번호 · 사용자 이름 입력
    Username(사용자 이름)은 영어로, 나중에 내 사이트 주소에 그대로 들어갑니다. 짧고 깔끔하게 지어 주세요. (예: hong-gildong)
  3. 이메일 인증
    가입한 이메일함에서 GitHub가 보낸 인증 코드를 확인해 입력합니다.
  4. 무료 플랜 선택
    요금제를 물으면 Free(무료)를 선택하면 됩니다. 결제 정보는 넣지 않아요.

✔️ 여기까지 되면

로그인 후 github.com/사용자이름 이 내 페이지예요. 이 사용자 이름을 꼭 기억하세요.

② Streamlit Community Cloud 가입

스트림릿은 파이썬 코드를 버튼·그래프가 있는 진짜 웹앱으로 바꿔 켜 주는 무료 서비스예요. 어렵지 않아요 — GitHub로 로그인만 하면 끝입니다.

  1. share.streamlit.io 접속
    주소창에 share.streamlit.io 를 칩니다.
  2. “Continue with GitHub” 클릭
    여러 로그인 버튼 중 GitHub로 계속하기를 누릅니다. (①에서 만든 계정)
  3. 권한 허용(Authorize)
    GitHub가 “Streamlit이 접근해도 될까요?”라고 물으면 Authorize(허용)를 누릅니다. 내 저장소를 앱으로 켜기 위한 연결이에요.
  4. 가입 완료
    이름·용도 등을 물으면 편하게 채우고 넘어갑니다. 대시보드 화면이 보이면 성공!

💡 왜 GitHub로 로그인하나요?

스트림릿은 GitHub 창고에 있는 코드를 가져다 앱으로 켜 줍니다. 그래서 두 계정이 연결돼 있어야 해요. 앞으로 “만들기 → GitHub에 올리기 → 스트림릿이 켜기”가 한 흐름이 됩니다.

③ AI 서비스 준비 (하나만)

코드를 대신 짜 줄 AI가 필요해요. 아래 셋 중 하나면 충분합니다. 이미 쓰는 게 있으면 그걸 쓰세요.

💡 어떤 걸 써도 괜찮아요

이 교재의 프롬프트(주문서)는 어떤 AI에 넣어도 되도록 썼어요. 강사 화면은 Claude로 보여드리지만, 여러분은 익숙한 것 하나만 준비하면 됩니다.

🎒 준비 완료 체크리스트

  • GitHub 가입 완료 — 내 사용자 이름을 안다
  • Streamlit Community Cloud에 GitHub로 로그인해 봤다
  • AI 서비스(셋 중 하나)에 로그인해 봤다
  • 세 계정의 이메일·비밀번호를 메모해 두었다
  • 노트북과 충전기를 챙긴다 (마우스 있으면 더 편해요)
