강의 전에 계정 3종 만들기

아래 3종 미리 준비 → 첫날 바로 제작 시작 · 전부 무료 · 설치 없음 · 브라우저(크롬 권장)만 필요

✅ 만들 계정 3개
  • ① GitHub — 결과물 보관·공개 (창고 + 집)
  • ② Streamlit Community Cloud — 코드 → 웹앱 실행 (GitHub로 로그인)
  • ③ AI 서비스 — 코드 작성 담당 (Claude · ChatGPT · Gemini 중 1개)
⚠️ 딱 하나만
  • 가입 이메일·비밀번호 메모 필수
  • ②는 반드시 ①과 같은 GitHub 계정으로 로그인
  • 세 계정 이메일 통일 → 편리

① GitHub 계정 만들기

  • GitHub = 파일 보관 + 그대로 웹사이트로 공개
  1. github.com 접속
    github.com → 우상단 Sign up
  2. 이메일 · 비밀번호 · Username 입력
    Username = 영어 · 내 사이트 주소에 그대로 사용 → 짧고 깔끔하게 (예: hong-gildong)
  3. 이메일 인증
    메일함 인증 코드 입력
  4. 무료 플랜 선택
    Free 선택 · 결제 정보 X
✔️ 여기까지 되면
  • 내 페이지 = github.com/사용자이름
  • 사용자 이름 꼭 기억

② Streamlit Community Cloud 가입

  • 스트림릿 = 파이썬 코드 → 버튼·그래프 있는 웹앱 (무료)
  • GitHub로 로그인만 하면 끝
  1. share.streamlit.io 접속
    share.streamlit.io
  2. "Continue with GitHub" 클릭
    ①에서 만든 계정으로
  3. 권한 허용(Authorize)
    접근 요청 → Authorize · 저장소를 앱으로 켜기 위한 연결
  4. 가입 완료
    이름·용도 입력 → 대시보드 표시 = 성공
💡 왜 GitHub로 로그인?
  • 스트림릿 = GitHub 창고의 코드를 앱으로 실행
  • 흐름: 만들기 → GitHub 올리기 → 스트림릿 켜기

③ AI 서비스 준비 (하나만)

  • 코드 작성용 AI 필요 · 아래 3종 중 1개면 충분 · 기존 사용 중이면 그대로
서비스주소메모
Claude (권장)claude.ai강의 예시 기준 · 무료 시작 가능
ChatGPTchatgpt.com익숙하면 이걸로
Geminigemini.google.com구글 계정이면 바로

  • 교재 프롬프트 = 어떤 AI에도 호환
  • 강사 화면 = Claude · 수강생 = 익숙한 것 1개
