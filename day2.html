  • 코드 짠 게 AI → 고치는 것도 AI 몫

1. 데이터를 읽는 눈 — 표를 열어보는 습관

  • 그래프보다 먼저 → 표를 눈으로 한 번 보기
  • 주문 딱 두 개만 · 외우지 말기
주문읽는 법무엇을 보나
df.head()“이 표(df)의 앞부분을 보여 줘”열 이름·생김새·값의 예시
df.describe()“이 표의 숫자 요약을 내 줘”개수·평균·퍼짐·최소·최대
  • describe() 표의 낯선 영어 → 뜻만 알면 끝
이름뜻
count값이 몇 개 있나 (빈 칸은 안 셈)
mean평균 (다 더해 개수로 나눈 값)
std퍼짐 정도 — 값들이 평균에서 얼마나 흩어져 있나
min / max가장 작은 값 / 가장 큰 값
🕵️ 의심하는 눈 — ‘평균의 함정’
  • 평균 = 큰 값 하나에 쉽게 끌려감
  • 나이 20, 21, 22, 23, 90 → 평균 35세 (실제론 대부분 20대)
  • 중앙값 = 22세 · 훨씬 정직
  • → 평균 볼 땐 "치우친 값 끼었나?" 함께 의심
  • ‘치우침’ 보려면 그림 필요 · 두 가지만 개념으로
  • 히스토그램 — 구간별 막대 높이로 "몇 개씩 몰렸나" · 봉우리 둘 = 두 무리
  • 상자그림(box plot) — 상자 = 중앙값·대다수 범위 · 바깥 점 = 튀는 값(이상치)
💡 오늘 여기선 개념만
  • 직접 그리는 코드 몰라도 됨
  • "AI야, 이 데이터로 히스토그램(상자그림) 그려줘"면 충분
  • 지금은 ‘무엇을 보는 그림인지’만

2. 인구 대시보드 만들기

  • 데이터 = 행정안전부 주민등록 인구 · 연령별 남녀 인구
  • 목표 = 동네 고르면 인구 피라미드 + 지표 카드 뜨는 스트림릿 앱
📦 오늘 쓸 데이터
  • 파일: population_2026_05.csv (연령별 주민등록 인구, cp949 인코딩)
  • 베이스: https://raw.githubusercontent.com/greatsong/modeudata-with-ai-examples/main/data/
  • 전체 주소: …/data/population_2026_05.csv → 프롬프트에 그대로 붙여넣기
📥 인구 데이터 출처 (행정안전부)

  • 큰 흐름: 연령별 인구현황 → 읍면동 → 1세 단위 → CSV 다운로드 · 자세한 절차는 교재 참고(화면은 바뀔 수 있음)
  • 빠른 실습 → 예제 저장소 population_2026_05.csv · 직접 받은 파일도 사용 가능
  • 오늘도 워크샵식 레벨업 · 한 계단씩

  2. LV2 · 데이터 읽기

  3. LV3 · 그래프

    총인구·평균연령·고령화율 지표 카드 얹어 완성
  • LV1~2는 어제 흐름 · 오늘은 LV3(피라미드)부터 주문서
① 샘플 프롬프트
스트림릿(streamlit) 앱 하나(main.py)를 만들어줘. '동네별 인구 피라미드'야.

- 데이터는 이 주소의 CSV를 pandas로 읽어:
  https://raw.githubusercontent.com/greatsong/modeudata-with-ai-examples/main/data/population_2026_05.csv
- 이 파일은 인코딩이 cp949(euc-kr)이고, 숫자에 천단위 쉼표(,)가 들어 있어.
  그래서 pd.read_csv(url, encoding="cp949", thousands=",") 로 읽어줘.
- 표는 행정구역(동네)별로 되어 있고, 0세부터 100세 이상까지 남자/여자 인구 열이 있어.
- 화면 위에 st.selectbox로 동네를 하나 고르게 해줘.
- 고른 동네의 '인구 피라미드'를 plotly 가로 막대그래프로 그려줘.
  남자는 왼쪽(값을 음수로), 여자는 오른쪽(양수)에 두고, 세로축은 나이.
- 한국어 라벨, 따뜻한 톤. 초보자용으로 주석을 한국어로 달아줘.
- main.py 전체 코드를 한 번에 줘.
  • AI가 완성된 코드 한 덩어리 반환 · 글자까지 똑같지 않아도 OK · 이런 모습이면 성공
② AI가 준 코드 (예시)
import streamlit as st
import pandas as pd
import plotly.graph_objects as go

st.title("🏘️ 동네별 인구 피라미드")

# 1) 데이터 불러오기 (cp949 인코딩, 천단위 쉼표 처리)
url = ("https://raw.githubusercontent.com/greatsong/"
       "modeudata-with-ai-examples/main/data/population_2026_05.csv")
df = pd.read_csv(url, encoding="cp949", thousands=",")

# 2) 동네(행정구역) 고르기
areas = df["행정구역"].tolist()
area = st.selectbox("동네를 골라 보세요", areas)
row = df[df["행정구역"] == area].iloc[0]

# 3) 나이별 남/여 인구 뽑기 (열 이름에서 나이만 골라 담기)
ages = list(range(0, 101))                 # 0세 ~ 100세 이상
male   = [row.get(f"남_{a}세", 0) for a in ages]
female = [row.get(f"여_{a}세", 0) for a in ages]

# 4) 피라미드 그리기 — 남자는 왼쪽(음수), 여자는 오른쪽(양수)
fig = go.Figure()
fig.add_bar(y=ages, x=[-m for m in male], name="남자", orientation="h")
fig.add_bar(y=ages, x=female,             name="여자", orientation="h")
fig.update_layout(barmode="relative", title=f"{area} 인구 피라미드",
                  xaxis_title="인구 수", yaxis_title="나이")

st.plotly_chart(fig, use_container_width=True)
  • 켜 보고 · 마음에 안 들면 대화로 개선 · 여기서도 세 박자
③ 프롬프트 개선 (예시)
좋아! 이제 여기에 지표 카드를 더해줘.
- 그래프 위에 st.metric 카드 3개를 나란히: 총인구, 평균연령, 고령화율(65세 이상 비율, %).
- 남자 막대는 파란색, 여자 막대는 분홍색으로.
- x축(인구 수)에 음수가 그대로 보이면 이상하니, 눈금 숫자는 절댓값으로 보여줘.
- 피라미드 아래에 '이 동네는 아이가 많은 편인가요, 어르신이 많은 편인가요?'
  한 줄 설명을 자동으로 붙여줘.
✔️ 지표 카드(st.metric)가 왜 중요한가?
  • 피라미드 = 모양 · 지표 카드 = 숫자 한 방 요약
  • 총인구·평균연령·고령화율(65세 이상) 셋만으로 "젊은/나이 든 동네" 판별
  • 그림 + 숫자 같이 봐야 데이터가 말을 검

3. 지도로 넓혀 보기 — 전국 고령화 단계구분도

  • 한 동네 → 이번엔 전국을 한눈에
  • 시군구별 고령화율을 색의 진하기로 칠한 지도 = 단계구분도(choropleth)
  • 색 진할수록 고령화 많이 진행
🗺️ 지도엔 ‘경계선’ 데이터가 하나 더
  • 인구 숫자만으론 지도 못 그림
  • 시군구 모양 담은 경계 데이터(GeoJSON) 별도 필요
  • 인구표(숫자) + 경계(모양)를 지역 코드로 짝지어 합치기
  • 짝짓기는 AI가 처리 → "코드로 합쳐줘"만 부탁
  • 경계 파일(GeoJSON) = 예제 저장소 제공 · 준비 방법은 교재 참고
① 샘플 프롬프트
스트림릿 앱(main.py)에 '전국 고령화 단계구분도'를 만들어줘.

- 인구 데이터는 이 CSV(cp949, thousands=","):
  https://raw.githubusercontent.com/greatsong/modeudata-with-ai-examples/main/data/population_2026_05.csv
- 시군구 경계는 전국 시군구 GeoJSON을 사용해줘.
- 시군구마다 '고령화율(65세 이상 인구 비율, %)'을 계산해줘.
- plotly의 choropleth(단계구분도)로, 고령화율이 높을수록 진한 색으로 전국 지도를 칠해줘.
- 지역을 맞출 때는 '지역 이름'이 아니라 '행정구역 코드'로 정확히 맞춰줘.
- 한국어 라벨, 마우스를 올리면 지역명과 고령화율이 보이게 해줘.
  • 지도 뜨면 색 진한 곳 눈으로 찾기 · 세 박자대로 ③ 개선
③ 프롬프트 개선 (예시)
좋아! 지도를 더 읽기 쉽게 고쳐줘.
- 고령화율을 5단계 구간(색 단계)으로 나눠서 칠해줘 (예: 10%·14%·20%·28% 기준).
- 고령화율이 가장 높은 시군구 5곳을 지도 옆에 표로 보여줘.
- 슬라이더로 '고령화율 몇 % 이상'만 골라 강조해서 볼 수 있게 해줘.
⏳ 지도는 오늘 ‘맛보기’
  • 경계 데이터까지 얽혀 조금 까다로움 · 완성 목표 아님
  • 색칠된 전국 지도 한 번 뜨면 충분 · 막히면 마지막 날 이어가기
